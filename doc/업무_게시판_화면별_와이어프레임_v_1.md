# 업무_게시판 — 화면별 와이어프레임(v2)

> 목적: 지침에 맞춘 최소 기능형(MVP) 웹앱의 화면 구성을 업데이트 버전으로 반영합니다. (조건 적용)

---

## 공통 레이아웃 (모든 페이지)

```
┌─────────────────────────────────────────────── 헤더 ───────────────────────────────────────────────┐
│  [업무_게시판]                               [로그인/내정보]                                     │
├──────────────────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                                  │
│                                          본문 영역                                              │
│                                                                                                  │
├─────────────────────────────────────────────── 푸터 ───────────────────────────────────────────────┤
│ © 2025 업무_게시판                                                                              │
└──────────────────────────────────────────────────────────────────────────────────────────────────┘
```

- 헤더: 브랜드/홈, 인증(로그인/로그아웃/마이페이지)
- 본문: 각 페이지별 컨텐츠
- 푸터: 최소 표시

---

## 1) 게시물 목록 (첫 화면)

```
[정렬: 최신순 ▼]   (로그인 시) [글쓰기]

┌──────────────────────────────────────────────────────────────────────────────────────────────────┐
│  게시물 카드 리스트 (페이지네이션)                                                                │
├──────────────────────────────────────────────────────────────────────────────────────────────────┤
│  #1 제목                                                      [댓글 12] [작성자 닉] [2025-08-20]  │
│  ──────────────────────────────────────────────────────────────────────────────────────────────   │
│  #2 제목                                                      [댓글 0]  [작성자 닉] [2025-08-19]  │
│  ...                                                                                              │
└──────────────────────────────────────────────────────────────────────────────────────────────────┘

[〈 이전]  1  2  3  [다음 〉]
```

- 비로그인: 목록/보기 가능, 글쓰기 버튼 클릭 시 로그인으로 이동
- 로그인: 글쓰기 버튼 노출 + 글쓰기/수정/삭제 권한 표시(본인 글에만 더보기 메뉴 표시)

## 5) 게시물 작성 / 수정 (편집 화면 동일 UI)

```
┌──────────────────────────────────── 게시물 작성 ────────────────────────────────────┐
│  제목 ▢▢▢▢▢▢▢                                                                       │
│  본문 (일반 텍스트 입력)                                                            │
│  ┌───────────────────────────────────────────────────────────────────────────────┐ │
│  │                                                                               │ │
│  │                          텍스트 입력 영역                                     │ │
│  │                                                                               │ │
│  └───────────────────────────────────────────────────────────────────────────────┘ │
│  [등록]   [취소]                                                                   │
└─────────────────────────────────────────────────────────────────────────────────────┘
```

- 권한: 로그인 사용자만 접근, 비로그인은 로그인 페이지로 이동
- 수정 모드: 상단에 "(수정)" 표시, 기존 데이터 바인딩

---

## 9) 사용자 흐름 요약

1. 첫 화면 = 게시물 목록
2. 글쓰기/댓글 작성 버튼 → 로그인 화면으로 이동(비로그인 시)
3. 로그인 완료 → 게시물 목록으로 리다이렉트
4. 로그인 사용자: 게시글/댓글 CRUD 가능, 비로그인: 목록/보기/댓글보기만
